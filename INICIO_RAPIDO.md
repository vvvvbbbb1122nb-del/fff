# Inicio Rapido - IPIAL

## Como Comecar (2 minutos)

### Opcao 1: Abrir Diretamente
1. Localize `index.html`
2. Clique duas vezes
3. Pronto!

### Opcao 2: Servidor Local
```bash
python -m http.server 8000
```
Acesse: http://localhost:8000

## Primeiros Passos

### 1. Explorar o Dashboard
- Veja estatisticas gerais
- Verifique atividade recente
- Analise distribuicao por curso

### 2. Adicionar Candidato
- Clique em "Candidatos"
- Clique em "Inscrever Candidato"
- Preencha os dados
- Faca upload da foto (opcional)
- Clique em "Guardar"

### 3. Gerar Relatorio
- Na lista de candidatos
- Clique em "Relatorio Geral"
- Arquivo HTML sera baixado

### 4. Publicar Resultados
- Va em "Publicacoes"
- Clique em "Publicar Resultados Agora"
- Acesse a pagina publica
